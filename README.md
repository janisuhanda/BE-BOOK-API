# book-api
Back End Rest API Server

Get Started
1. Sesuaikan konfigurasi database nya
2. Create database name
3. npm install
4. node .\src\app.js

akses sesuai route :
get /
get /api/books #all
post /api/books #create
get /api/books/:id #findOne
put /api/books/:id #update
delete /api/books/:id #delete
